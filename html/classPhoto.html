Detailed Description

Represents a photograph with location metadata.

The Photo class extends Multimedia to represent photograph objects with support for geographic coordinates (latitude and longitude). This enables storing and retrieving location information for photos, useful for location-based organization and display. Photos are created and managed through the Manager class.

Note
Photos are immutable after creation (coordinates set via constructor)
