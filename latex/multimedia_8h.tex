\doxysubsection{Detailed Description}
Header file for the \doxylink{classMultimedia}{Multimedia} base class. 

This file defines the abstract \doxylink{classMultimedia}{Multimedia} class which serves as the base class for all multimedia types (\doxylink{classPhoto}{Photo}, \doxylink{classVideo}{Video}, \doxylink{classFilm}{Film}). It provides common functionality for managing multimedia properties and defines the interface that all multimedia objects must implement.
