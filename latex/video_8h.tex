Header file for the \doxylink{classVideo}{Video} class. 

This file defines the \doxylink{classVideo}{Video} class, which represents a video multimedia object. \doxylink{classVideo}{Video} extends the \doxylink{classMultimedia}{Multimedia} base class with duration information and provides functionality for managing video playback and serialization. The \doxylink{classVideo}{Video} class serves as a base class for specialized video types like \doxylink{classFilm}{Film}.
