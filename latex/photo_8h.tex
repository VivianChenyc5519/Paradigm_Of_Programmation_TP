\doxysubsection{Detailed Description}
Header file for the \doxylink{classPhoto}{Photo} class. 

This file defines the \doxylink{classPhoto}{Photo} class, which represents a photograph multimedia object. \doxylink{classPhoto}{Photo} extends the \doxylink{classMultimedia}{Multimedia} base class with geographic location information (latitude and longitude) allowing photos to be tagged with their capture location.
