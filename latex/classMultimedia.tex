\doxysubsection{Detailed Description}
Abstract base class for all multimedia objects. 

The \doxylink{classMultimedia}{Multimedia} class is an abstract base class that defines the interface and common properties for all multimedia types. It stores fundamental information like name and file path, and defines pure virtual methods that must be implemented by derived classes (\doxylink{classPhoto}{Photo}, \doxylink{classVideo}{Video}, \doxylink{classFilm}{Film}). The \doxylink{classManager}{Manager} class has friend access to control object creation.

\begin{DoxyNote}{Note}
This is an abstract class with pure virtual methods and should not be instantiated directly. Use derived classes instead.
\end{DoxyNote}
