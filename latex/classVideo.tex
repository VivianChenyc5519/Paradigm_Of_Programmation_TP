\doxysubsection{Detailed Description}
Represents a video multimedia object with duration. 

The \doxylink{classVideo}{Video} class extends \doxylink{classMultimedia}{Multimedia} to represent video objects with support for duration information. It provides methods for getting/setting the video duration and implements playback and serialization functionality. The class is designed to be inherited by more specialized video types (such as \doxylink{classFilm}{Film}) and is managed through the \doxylink{classManager}{Manager} class.

\begin{DoxyNote}{Note}
Duration is stored in seconds 

This class has protected constructors and member visibility to allow derivation by specialized video types
\end{DoxyNote}
