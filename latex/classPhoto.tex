\doxysubsection{Detailed Description}
Represents a photograph with location metadata. 

The \doxylink{classPhoto}{Photo} class extends \doxylink{classMultimedia}{Multimedia} to represent photograph objects with support for geographic coordinates (latitude and longitude). This enables storing and retrieving location information for photos, useful for location-\/based organization and display. Photos are created and managed through the \doxylink{classManager}{Manager} class.

\begin{DoxyNote}{Note}
Photos are immutable after creation (coordinates set via constructor)
\end{DoxyNote}
