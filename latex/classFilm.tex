\doxysubsection{Detailed Description}
Represents a film with chapter management capabilities. 

The \doxylink{classFilm}{Film} class extends \doxylink{classVideo}{Video} to add support for film chapters. It allows storing and retrieving chapter information (chapter lengths) for a video file, enabling chapter-\/based navigation and display. Films are managed through the \doxylink{classManager}{Manager} class which controls their creation and lifecycle.

\begin{DoxyNote}{Note}
Chapter data is managed dynamically and properly copied during object duplication to ensure safe memory management.
\end{DoxyNote}
